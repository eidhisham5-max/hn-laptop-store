## Tablet (768px - 1023px)

### Layout Adjustments
- Cart items: Full width, stacked
- Order summary: Below cart items
- Coupon code: Simplified layout
- Delivery options: Stacked vertically

## Mobile (320px - 767px)

### Layout Adjustments
- Cart items: Single column, simplified
- Order summary: Sticky at bottom
- Quantity controls: Touch-friendly
- Remove buttons: Larger touch targets

## Interactive States
- **Quantity Changes**: Real-time price updates
- **Remove Items**: Confirmation dialog
- **Coupon Code**: Validation and application
- **Delivery Options**: Price updates
- **Checkout Button**: Loading state
- **Empty Cart**: Call-to-action to browse products
