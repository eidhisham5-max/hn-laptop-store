## Tablet (768px - 1023px)

### Layout Adjustments
- Products list: Full width table
- Add/Edit form: Single column layout
- Actions: Dropdown menu
- Images: Grid layout

## Mobile (320px - 767px)

### Layout Adjustments
- Products list: Card layout
- Add/Edit form: Single column
- Actions: Touch-friendly buttons
- Images: Horizontal scroll

## Interactive States
- **Product List**: Hover effects, sorting, filtering
- **Add/Edit Form**: Real-time validation, auto-save
- **Image Upload**: Drag & drop, preview
- **Status Toggle**: Quick enable/disable
- **Bulk Actions**: Select multiple products
- **Search**: Real-time filtering
