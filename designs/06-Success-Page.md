## Tablet (768px - 1023px)

### Layout Adjustments
- Success message: Centered layout
- Order details: Full width
- Shipping info: Below order details
- What's next: Stacked vertically

## Mobile (320px - 767px)

### Layout Adjustments
- Success message: Single column
- Order details: Simplified list
- Shipping info: Compact layout
- What's next: Collapsible sections
