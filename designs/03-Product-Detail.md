## Tablet (768px - 1023px)

### Layout Adjustments
- Product gallery: Stacked vertically
- Product info: Full width below gallery
- Specifications: 2-column layout
- Reviews: Simplified layout
- Related products: 2-column grid

## Mobile (320px - 767px)

### Layout Adjustments
- Product gallery: Horizontal scroll
- Product info: Single column
- Specifications: Single column
- Reviews: Simplified, stacked
- Related products: Single column

### Mobile Product Gallery

## Interactive States
- **Image Gallery**: Thumbnail navigation, zoom on hover
- **Quantity Selector**: Increment/decrement buttons
- **Add to Cart**: Loading state, success feedback
- **Reviews**: Expandable review cards
- **Related Products**: Hover effects, quick add to cart
- **Specifications**: Expandable sections
- **Stock Status**: Real-time updates
