## Tablet (768px - 1023px)

### Layout Adjustments
- Checkout form: Full width
- Order summary: Below form
- Steps: Horizontal progress bar
- Form fields: Stacked layout

## Mobile (320px - 767px)

### Layout Adjustments
- Checkout form: Single column
- Order summary: Collapsible
- Steps: Vertical progress indicator
- Form fields: Full width
