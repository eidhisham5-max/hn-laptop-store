### Layout Adjustments

- Status updates: Dropdown menu
- Actions: Touch-friendly buttons

- Order details: Full screen modal

## Interactive States
- **Order List**: Sorting, filtering, bulk actions
- **Status Updates**: Real-time status changes
- **Order Details**: Expandable sections
- **Timeline**: Interactive progress tracking
- **Notes**: Add/edit order notes
- **Export**: CSV/PDF export options
